# svg-autocrop

[![Build](https://github.com/gsmlg-ci/svg-autocrop/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/svg-autocrop/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/svg-autocrop`
- `ghcr.io/gsmlg-dev/svg-autocrop`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/svg-autocrop:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/svg-autocrop:latest
```

## Build

```bash
docker build -t gsmlg/svg-autocrop .
```
